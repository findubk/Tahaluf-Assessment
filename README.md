## Commands

1. Install dependencies:

    ```sh
    npm install
    ```

2. Run development server:

    ```sh
    npm run dev
    ```

3. Create a production build:

    ```sh
    npm run build
    ```

4. Serve the production build:

    ```sh
    npm run preview
    ```

5. Run tests:

    ```sh
    npm run test
    ```

6. Run tests with coverage:

    ```sh
    npm run coverage
    ```
